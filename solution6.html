Difference between HTML and HTML5

HTML HTML5
It does not provide native support for embedding audio and video content. It requires external plugins for this. HTML5 introduces native <audio> and <video> tags for embedding audio and video content directly into web pages.
HTML relies on generic tags like <div> and <span> to structure content. HTML5 introduces new semantic elements like <header>, <nav>, <main>, <aside>, <article>, and <footer> to better define the structure and meaning of web content.
HTML's capabilities are primarily limited to structuring content and linking to other web pages. HTML5 expands the possibilities of web development with APIs for geolocation, drag-and-drop functionality, canvas drawing, web storage, and more.
